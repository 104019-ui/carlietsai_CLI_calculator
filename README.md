# CLI calculator
A simple calculator you can run in Python to do basic math like adding, subtracting, multiplying, and dividing.

## Description
The calculator runs in the terminal and walks you through each step of a calculation. It helps you see how Python functions work and how to handle user input, making it an easy way to practice coding skills.

## How to Run
1. Download or clone this repository.  
2. Open the "CLI_calculator.py" file in Python.  
3. Follow the prompts in the terminal to perform your calculations.

## Features
- Add numbers  
- Subtract numbers  
- Multiply numbers  
- Divide numbers
